# 인증 프로세스

## OAuth 제공자
- Google OAuth
- Apple OAuth

## 회원가입 프로세스

### 1단계: OAuth 인증
사용자가 Google 또는 Apple 계정으로 OAuth 인증 진행

### 2단계: 추가 정보 입력
OAuth 인증 성공 후, 다음 정보를 입력받아 저장:
- 이메일
- 이름
- 기수

### 3단계: 가입 완료
사용자 정보 저장 완료
- 상태: "승인 대기"

### 4단계: 관리자 승인
관리자가 사용자를 승인해야 로그인 가능
- 승인 전: 로그인 불가
- 승인 후: 서브노트 조회 가능

## 사용자 상태
1. **승인 대기 (Pending)**: 가입 완료, 관리자 승인 전
2. **승인 완료 (Approved)**: 관리자 승인, 로그인 및 서브노트 조회 가능
3. **거부 (Rejected)**: 관리자가 승인 거부

## 권한 레벨
1. **수강생 (Student)**: 서브노트 조회만 가능
2. **관리자 (Admin)**: 서브노트 CRUD, 사용자 승인 관리
